# React eCommerce

## Instalar proyecto

```shell
git clone https://github.com/joshuacba08/react-ecommerce-ibec-mar2023.git
cd react-ecommerce-ibec-mar2023
npm install
```


## Arrancar proyecto

```shell
npm run dev
```
